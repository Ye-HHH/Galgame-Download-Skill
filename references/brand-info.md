# 会社说明文件规范

每个会社文件夹下放一个说明 txt 文件。

## 文件名

三语命名：`中文俗称_日文名_英文名.txt`

示例：
- `九路破_クルッポ_Qruppo.txt`
- `甜蜜社_スミー_SMEE.txt`
- `海豹社_あざらしそふと_AzarashiSoft.txt`

⚠️ CJK 文件名必须通过 Python 文件写入（bash 传参会乱码）。

## 文件内容

```markdown
会社名称：中文俗称 / 日文名 / 英文名

代表作：中文名 / 日文名 / 英文名

风格特点：（一段话概括会社的风格、擅长领域、目标受众）

═══════════════════════════════════

【已下载游戏】

■ 中文名 / 日文名 / 英文名
  内容简介（1-3句，讲清楚故事）
  特点：核心卖点、评分、时长等关键信息。
```

## 禁止事项

- ❌ 不要写解压密码
- ❌ 不要写下载来源
- ❌ 文件名不要只用英文
